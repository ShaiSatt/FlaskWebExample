Flash Messages

לומדים כיצד להציג הודעות חכמות למשתמשים — הצלחה, שגיאה, אזהרה — שנעלמות מעצמן

שלב 1 מתוך 7

💬 מה זו הודעת Flash?

כשמשתמשים מתחברים לאתר, נרשמים, או עושים פעולה כלשהי — הם מצפים לקבל תגובה. Flash היא הודעה חד-פעמית שמוצגת בדף הבא ואז נעלמת לנצח.

⚡ דוגמאות מהפרויקט שלנו

!נכנסת בהצלחה ✓
flash(" !נכנסת בהצלחה", "success")
משתמש לא קיים johndoe ✗
flash(f"משתמש לא קיים {user}", "error")
!alice נרשמת בהצלחה ✓
flash(f"{username} נרשמת בהצלחה", "success")

🔑 למה "Flash"?

כמו הבזק של מצלמה — מאיר לרגע ונעלם. ההודעה נשמרת ב-Session (זיכרון הצד-שרת) ומוצגת פעם אחת בלבד. ברגע שהדפדפן קרא אותה — היא נמחקת.

⚠️
בלי Flash: אחרי redirect, כל המשתנים של הפונקציה נמחקים. לא ניתן לשלוח מידע לדף הבא ישירות — Flash פותרת בדיוק את הבעיה הזו!

💡 מה נלמד בלומדה זו?

  • כיצד שולחים הודעת Flash מ-Python עם flash()
  • כיצד מציגים הודעות Flash ב-base.html
  • ההבדל בין קטגוריה "success" ל-"error"
  • כיצד CSS מעצב כל קטגוריה בנפרד
  • הקוד JavaScript שגורם להודעה להיעלם אחרי 3 שניות

🐍 שליחת Flash מ-main.py

שלב ראשון: שולחים הודעה מ-Python. הפונקציה flash() מקבלת שני פרמטרים:

flash("הטקסט", "קטגוריה")
הטקסט — מה שיוצג למשתמש. יכול להכיל משתנים עם f-string.
"success"
קטגוריה — שם חופשי שאנחנו בוחרים. משמש לעיצוב CSS (ירוק לhצלחה, אדום לשגיאה).
  main.py — שלוש דוגמאות מהפרויקט
# ── דוגמה 1: login מצליח ──
if user in valid_users:
    current_user = user
    flash(" !נכנסת בהצלחה", "success")
    return redirect(url_for('home'))

# ── דוגמה 2: login נכשל ──
else:
    flash(f"משתמש לא קיים {user}", "error")

# ── דוגמה 3: signup מצליח ──
valid_users.append(username)
flash(f"{username} נרשמת בהצלחה", "success")
return redirect(url_for('login'))

# ── דוגמה 4: logout ──
flash(f"התנתקת בהצלחה! {current_user}", "success")
current_user = 'Guest'
return redirect(url_for('home'))

⚙️ תנאי הכרחי — secret_key

Flash משתמשת ב-Session, שדורשת מפתח הצפנה. בלי זה Flask יקרוס!

app = Flask(__name__)
app.secret_key = "megido_secret_key"  ← חובה!
🔐
secret_key משמש להצפנת ה-Session. בפרויקט אמיתי — שומרים אותו בנפרד ולא ב-קוד.

📄 הצגת Flash ב-base.html

שלב שני: מציגים את ההודעות בתבנית. הקוד נמצא ב-base.html — ולכן מופיע בכל דף!

  templates/base.html — קטע ה-Flash
<main class="container">

    <!-- הצגת הודעות Flash -->
    {% with messages = get_flashed_messages(with_categories=true) %}
        {% if messages %}
            {% for category, message in messages %}
                <div class="flash-box {{ category }}">
                    {{ message }}
                </div>
            {% endfor %}
        {% endif %}
    {% endwith %}

    {% block content %}{% endblock %}
</main>

🔍 פירוק שורה שורה

get_flashed_messages(with_categories=true)
פונקציה של Flask שמחזירה את כל ההודעות שנשמרו ב-Session. עם with_categories=true מקבלים גם את שם הקטגוריה.
{% with messages = ... %}
שומרים את תוצאת הפונקציה במשתנה messages — בתוך הבלוק הזה בלבד.
{% for category, message in messages %}
לולאה שמפרקת כל פריט לשניים: category (כמו "success" או "error") ו-message (הטקסט).
class="flash-box {{ category }}"
הקלאס של ה-div יהיה "flash-box success" או "flash-box error" — כך CSS מעצב כל אחת בנפרד!

🎨 עיצוב ב-CSS

הקטגוריה הופכת לקלאס CSS — וכל קטגוריה מקבלת עיצוב שונה. כך נראה ב-style.css:

  static/css/style.css — עיצוב Flash
/* תיבת הבסיס — משותפת לכל הקטגוריות */
.flash-box {
    width: 40%;
    margin: 20px auto;        /* מרכז אופקית */
    padding: 12px;
    border: 2px solid black;
    border-radius: 8px;
    text-align: center;
    font-weight: bold;
}

/* ירוק — הצלחה */
.success {
    background-color: #d4edda;  /* ירוק בהיר */
    color: #155724;             /* ירוק כהה */
}

/* אדום — שגיאה */
.error {
    background-color: #f8d7da;  /* אדום בהיר */
    color: #721c24;             /* אדום כהה */
}

⏱️ הדעיכה האוטומטית — JavaScript

ב-base.html יש קוד JavaScript שגורם להודעה להיעלם אחרי 3 שניות:

  templates/base.html — JavaScript
<script>
    setTimeout(function() {
        document.querySelectorAll('.flash-box').forEach(function(el) {
            el.style.transition = "opacity 0.8s";
            el.style.opacity = "0";
        });
    }, 3000);  ← 3000 מילישניות = 3 שניות
</script>

🔍 פירוק הקוד

setTimeout(function, 3000)
מריץ פונקציה אנונימית אחרי עיכוב של 3000 מילישניות (= 3 שניות).
querySelectorAll('.flash-box')
מוצא את כל האלמנטים עם קלאס flash-box בדף — יכולות להיות כמה הודעות בו-זמנית.
transition = "opacity 0.8s"
מוסיף אפקט דעיכה חלקה של 0.8 שניות — במקום שההודעה תיעלם בבת אחת.
opacity = "0"
הופך את ההודעה לשקופה לחלוטין. שימו לב: ההודעה עדיין קיימת ב-HTML אבל בלתי נראית!

🗺️ המסע המלא של הודעת Flash

מרגע שמשתמש שולח טופס Login ועד שההודעה נעלמת — הנה כל השלבים:

1
המשתמש שולח טופס Login
דפדפן שולח POST /login עם שם המשתמש
2
Flask מריץ את login() ב-main.py
בודק אם המשתמש קיים ב-valid_users
3
flash() שומרת הודעה ב-Session
flash(" !נכנסת בהצלחה", "success") — נשמר בזיכרון השרת
4
redirect() מפנה לדף הבית
return redirect(url_for('home')) — שולח דפדפן ל- GET /
5
get_flashed_messages() שולפת ומוחקת
Jinja2 שולפת את ההודעות מה-Session — ומוחקת אותן מיד!
6
הדפדפן מציג את home.html עם ההודעה
ה-div עם class="flash-box success" מקבל עיצוב ירוק
7
JavaScript מדעיך אחרי 3 שניות
