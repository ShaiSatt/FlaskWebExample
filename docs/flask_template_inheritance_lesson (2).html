💡 מה נלמד בלומדה זו?

  • מה זה base.html ולמה הוא שימושי
  • כיצד להשתמש ב-{% block %} לאזורים מתחלפים
  • כיצד כל דף "יורש" מהבסיס בעזרת {% extends %}
  • ניתוח הקוד האמיתי מפרויקט Flask של בית הספר

🔑 שלושת הפקודות הקסמיות

Flask משתמש בשפת תבניות בשם Jinja2. פקודותיה נכתבות בתוך {% ... %} (לוגיקה) ו-{{ ... }} (הצגת ערכים).

לירושת דפים יש שלוש פקודות מרכזיות:

{% extends "base.html" %}
בראש כל דף-בן — "הדף הזה יורש מ-base.html". חייבת להיות השורה הראשונה בקובץ!
{% block name %} ... {% endblock %}
ב-base.html — מגדיר "חלון" בשם name. כל דף-בן יכול למלא אותו בתוכן אחר.
{% block name %} ... {% endblock %}
בדפי הבן — ממלאים את ה"חלון" בתוכן ייחודי. מה שמחוץ לבלוקים לא יוצג!

📄 base.html — התבנית האם

זהו הקוד האמיתי מהפרויקט. קוראים אותו בעיון ומאתרים כל {% block %} וכל {{ משתנה }}:

  templates/base.html
<!DOCTYPE html>
<html>
<head>
    <title>{% block title %}Flask App{% endblock %}</title>   ← block ראשון: שם הדף בלשונית
    <link rel="stylesheet" href="/static/css/style.css">
    <link rel="stylesheet" href="https://cdnjs.cloudflare.com/.../font-awesome.min.css">
    <link href="https://cdn.jsdelivr.net/npm/[email protected]/.../bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <header>
        <img class="logoimg" src="/static/images/nivut.jpg" alt="our logo">
        <h4>Welcome {{ user }} !</h4>   ← משתנה שמגיע מ-Python
        <nav><ul>
            <li><a href="/"><i class="fas fa-home"></i> Home</a></li>
            <li><a href="/users"><i class="fas fa-users"></i> Users</a></li>
            {% if user == 'Guest' %}   ← תפריט דינמי! משתנה לפי ההתחברות
                <li><a href="/login"><i class="fas fa-sign-in-alt"></i> Login</a></li>
                <li><a href="/signup"><i class="fas fa-user-plus"></i> Signup</a></li>
            {% else %}
                <li><a href="/restricted">Restricted</a></li>
                <li><a href="/logout">Logout</a></li>
            {% endif %}
        </ul></nav>
    </header>

    <!-- הודעות Flash -->
    {% with messages = get_flashed_messages(with_categories=true) %}
        {% if messages %}
            {% for category, message in messages %}
                <div class="flash-box {{ category }}">{{ message }}</div>
            {% endfor %}
        {% endif %}
    {% endwith %}

    <main class="container">
        {% block content %}   ← block שני: תוכן ייחודי לכל דף
        {% endblock %}
    </main>

    <footer>
        <p>&copy; 2026 Flask App Demo - Megido High School</p>
    </footer>
</body></html>

🔎 מה מגדיר ה-base.html?

block title — שם הדף בלשונית הדפדפן. ברירת מחדל היא "Flask App", כל דף יכול לדרוס אותה.

block content — האזור הראשי. זה מה שמשתנה בין הדפים.

כל שאר הקוד — header, nav, flash messages, footer — נכתב פעם אחת ומשותף לכולם.

⭐
שימו לב: {% if user == 'Guest' %} גורם לתפריט להשתנות אוטומטית — אורחים רואים Login/Signup, משתמשים מחוברים רואים Restricted/Logout.

👥 users.html — עם לולאה

  templates/users.html
{% extends "base.html" %}

    <h1>Users</h1>
    <table class="userstable">

🐍 main.py — הדבק שמחבר הכל

Flask מחבר בין כתובת URL לדף HTML באמצעות render_template(). כך נראה main.py בפרויקט:

  main.py
from flask import Flask, render_template, request, redirect, url_for, flash

app = Flask(__name__)
app.secret_key = "megido_secret_key"
valid_users = []           # רשימת משתמשים רשומים
current_user = 'Guest'   # משתמש נוכחי — מתחיל כ-Guest

@app.route('/')
def home():
    return render_template('home.html', user=current_user)
    # מעביר את current_user לתבנית בשם "user"
    # base.html יציג: Welcome Guest !

@app.route('/login', methods=['GET', 'POST'])
def login():
    global current_user
    if request.method == 'POST':
        user = request.form['username']
        if user in valid_users:
            current_user = user
            flash(" !נכנסת בהצלחה", "success")
            return redirect(url_for('home'))
        else:
            flash(f"משתמש לא קיים {user}", "error")
    return render_template('login.html', user=current_user, message='')

@app.route('/restricted')
def restricted():
    if current_user != 'Guest':
        return render_template('restricted.html', user=current_user)
    else:
        return '<h1> access denied </h1>'  ← HTML גולמי, ללא base!

@app.route("/users")
def show_users():
    return render_template("users.html", user=current_user, users=valid_users)
    # מעביר שני פרמטרים: user לתפריט, users לטבלה

📚 סיכום — מה למדנו

  • base.html מכיל את השלד המשותף: header עם לוגו, תפריט דינמי, flash messages ו-footer
  • {% block title %} ו-{% block content %} מגדירים אזורים שדפי הבן ממלאים
  • {% extends "base.html" %} בראש הדף — יורשים מהבסיס
  • render_template() ב-main.py מחבר URL לתבנית ומעביר נתונים
  • נתונים שנשלחים (כמו user) זמינים גם בדף הבן וגם ב-base.html
